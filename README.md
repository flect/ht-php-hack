ht-php-hack
===========
Sample project that use Hacklang(Facebook Hack) for Heroku training.

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-php-hack.git
$ cd ht-php-hack/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application
```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
